# IntelliJ IDEA Cheat Sheet (Linux/Windows)

IntelliJ IDEA - bu Java va boshqa tillar uchun kuchli IDE. Quyida eng ko'p ishlatiladigan klaviatura yorliqlari (shortcuts) keltirilgan. Ushbu yorliqlar Linux va Windows uchun moslashtirilgan .

---

## 1. Umumiy (General)

| Qisqa yo'l | Vazifasi |
|-----------|----------|
| `Alt + [0-9]` | Tegishli vosita oynasini ochish (masalan, `Alt + 1` - Project)  |
| `Ctrl + S` | Hammasini saqlash  |
| `Ctrl + Alt + Y` | Sinxronlash  |
| `Ctrl + Shift + F12` | Editorni butun ekran qilish (yoki normal holatga qaytarish)  |
| `Ctrl + Alt + S` | Sozlamalarni (Settings) ochish  |
| `Ctrl + Alt + Shift + S` | Loyiha strukturasini (Project Structure) ochish  |
| `Ctrl + Shift + A` | Har qanday amalni (action) nomi bo'yicha qidirish  |
| `Ctrl + Tab` | Ochiq fayllar va vosita oynalari o'rtasida almashish  |
| `Shift + F4` | Joriy faylni yangi oynada ochish  |

## 10. Tez-tez ishlatiladigan Live Templates

Live Templates - bu tez kod yozish uchun qisqa yorliqlar .

| Shablon | Natija |
|---------|--------|
| `psvm` | `public static void main(String[] args) {}` |
| `sout` | `System.out.println();` |
| `soutv` | `System.out.println("var = " + var);` |
| `fori` | `for (int i = 0; i < ; i++) {}` |
| `itar` | `for (int i = 0; i < array.length; i++) {}` |
| `iter` | Java 5 foreach sikli |
| `itli` | `for (int i = 0; i < list.size(); i++) {}` |
| `itco` | `for (Object o : collection) {}` |
| `psf` / `psfi` / `psfs` | `public static final` / `public static final int` / `public static final String` |
| `thr` | `throw new` |
| `inst` | `if (variable instanceof Type) {}` |

## 🔧 Qisqa Yo'llarni Sozlash

Agar biror qisqa yo'l ishlamasa yoki o'zgartirmoqchi bo'lsangiz:

1.  `Ctrl + Alt + S` → **Keymap** bo'limiga o'ting .
2.  Qidiruv maydoniga kerakli amal nomini yozing (masalan, "Rename").
3.  Amalni o'ng tugmasini bosib, **Add Keyboard Shortcut** orqali yangi kombinatsiya belgilang .
4.  Agar asl sozlamalarga qaytmoqchi bo'lsangiz, **Restore Defaults** tugmasidan foydalaning .
