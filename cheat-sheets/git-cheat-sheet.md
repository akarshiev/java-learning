# Git Komandalari Qo'llanmasi

## Asosiy Komandalar

| Komanda | Tavsif |
|---------|--------|
| `git init` | Yangi repository yaratish |
| `git clone <url>` | Repository klonlash |
| `git status` | Holat ko'rish |
| `git add .` | Barcha o'zgarishlarni staging ga qo'shish |
| `git commit -m "message"` | O'zgarishlarni saqlash |
| `git push` | Remote ga yuklash |
| `git pull` | Remote dan olish |
| `git branch` | Branchlar ro'yxati |
| `git checkout -b <branch>` | Yangi branch yaratish |
| `git merge <branch>` | Branch birlashtirish |
| `git log` | Tarix ko'rish |
