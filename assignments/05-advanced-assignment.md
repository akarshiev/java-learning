# 5-Modul Topshiriqlari: Advanced Java

## Topshiriqlar

### 1. Stream API
Stream API yordamida ma'lumotlarni filtrlash, saralash va transformatsiya qiling.

### 2. Maven Project
Maven yordamida yangi Java loyiha yarating va dependency qo'shing.

### 3. HTTP Client
Java HTTP Client yordamida public API dan ma'lumot oling.

### 4. GSON
JSON faylni o'qib, ob'ektga aylantiring (deserialization).

### 5. Reflection API
Reflection yordamida klass maydonlari va metodlari ro'yxatini chiqaring.

### 6. Custom Annotation
O'zingizning annotatsiyangizni yarating va Reflection yordamida foydalaning.
